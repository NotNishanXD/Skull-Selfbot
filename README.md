# Skull-Selfbot
N
Chat-Packing Selfbot.
# How to use?
1. Clone this respository by running:
```bash
git clone https://github.com/NotNishanXD/Skull-Selfbot.git
```
2. Run:
```bash
cd Skull-Selfbot
```
3. Install dependies by running:
```bash
pip install -r requirements.txt
```
4. Run the selfbot:
```bash
python skull.py
```
